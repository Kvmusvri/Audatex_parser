Документация Audotex Parser

Обзор проекта

Audotex Parser - система парсинга данных автомобилей с сайта Audatex с использованием Selenium WebDriver.

Архитектура системы

Система состоит из следующих основных модулей:

  • auth - Аутентификация и управление пользователями

  • database - Модели данных и работа с базой данных

  • parser - Основной модуль парсинга данных

  • queue - Обработка очереди заявок

  • security - Система безопасности и мониторинга
